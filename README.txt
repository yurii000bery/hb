Сайт-привітання для Дідуся

Файли:
- index.html — головна сторінка з привітанням і двома папками.
- mi-tut.html — перша папка, місце для відео / галереї / повідомлень.
- dlya-tebe.html — друга папка, місце для фінального подарунку.
- styles.css — вся стилізація.
- script.js — мікроанімації та логіка відкриття папок по черзі.
- assets/ — ілюстрація родини та favicon.

Щоб редагувати контент, змінюй тексти всередині HTML-файлів. Найважливіші місця: блок media-slot у mi-tut.html і блок gift-certificate / gift-details у dlya-tebe.html.

PNG папки на головній:
- img/mi_tut_folder.png — картинка для папки “Ми тут”
- img/dlya_tebe.png — картинка для папки “Для тебе”

Якщо замінюєш картинки вручну, збережи ті самі назви файлів і шлях img/. Код уже підключений.
